Um projeto de estudo

Este é um pequeno projeto com o intuito de desenvolver habilidades de desenvolvimento Front End. Com este projeto busquei entender melhor o processo de requisições http, utilização de funções assíncronas e a implementação de promises. Anteriormente a este projeto, já havia desenvolvido um trabalho semelhante, utilizando React. Contudo, entendi que utilizar as linguagens em suas formas nativas, para realizar algo parecido, poderia trazer vantagens ao agregar e solidificar o conhecimento dos processos.
